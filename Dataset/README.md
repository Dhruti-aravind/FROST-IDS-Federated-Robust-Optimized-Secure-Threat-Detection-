UNSW-NB15 (Training Dataset)
A modern intrusion detection dataset generated using real and synthetic network traffic, containing nine attack categories along with normal flows. It includes rich flow-based features extracted from ~100GB of traffic, making it suitable for training robust IDS models.

Link: [https://www.kaggle.com/datasets/dhoogla/unswnb15/versions/5](https://www.kaggle.com/datasets/dhoogla/unswnb15/versions/5)

CICIDS2017 / CSE-CICIDS2018 (External Testing Dataset)
A realistic benchmark dataset with labeled network flows and full packet captures, covering diverse attack scenarios and user behaviors. It is widely used to evaluate model generalization on unseen, real-world-like intrusion patterns.

Link: [https://www.kaggle.com/datasets/ernie55ernie/improved-cicids2017-and-csecicids2018](https://www.kaggle.com/datasets/ernie55ernie/improved-cicids2017-and-csecicids2018)
